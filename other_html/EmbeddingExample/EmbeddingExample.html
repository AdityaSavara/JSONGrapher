Embedding Instructions:

    To embed an interactive graph like this on your own website, follow the below steps.

  1. Copy and insert the following code, including the div and script tags. Update the jsonUrlString with your JSON file's URL.
  2. Give your <div> element a unique ID (such as graph2) and also put that name into the loadMergeandPlotData arguments.
  3.     <!-- Start of JSONGrapher Embedding Code -->
    <div id="graph2"></div>
    <script type="module">
      import { loadMergeAndPlotData } from 'https://cdn.jsdelivr.net/gh/AdityaSavara/JSONGrapher@main/src/index.js';
      let jsonUrlString = "https://github.com/AdityaSavara/JSONGrapherExamples/blob/main/ExampleDataRecords/2-Differential_Heats_of_Adsorption/SrTiO3_rainbow.json";
      const messagesToUserDiv = document.getElementById("messagesToUserDiv");
      const errorMessagesDiv = document.getElementById("errorMessagesDiv");
      loadMergeAndPlotData(null, jsonUrlString, "url", graph2, messagesToUserDiv, errorMessagesDiv);
    </script>
    <div id="messagesToUserDiv"></div>
    <div id="errorMessagesDiv"></div>
    <!-- End of JSONGrapher Embedding Code -->
