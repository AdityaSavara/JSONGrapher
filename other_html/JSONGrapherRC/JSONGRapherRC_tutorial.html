JSONGrapherRC Tutorial

Step 1: Preparing to Create a Record

We will plot the height of a pear tree per year, with some variation.

x_label_including_units = "Time (years)"
y_label_including_units = "Height (m)"
time_in_years = [0, 1, 2, 3, 4]
tree_heights = [0, 0.42, 0.86, 1.19, 1.45]

Step 2: Creating a New JSONGrapher Record

Use create_new_JSONGrapherRecord() to create a new record.

try: 
    from JSONGRapherRC import JSONRecordCreator
except:
    import JSONRecordCreator

Record = JSONRecordCreator.create_new_JSONGrapherRecord()
print(Record)
Record.add_hints()
Record.print_to_inspect()

Expected Output:

Warning: Printing directly will return the raw record without some automatic updates.
{
    "comments": "",
    "datatype": "",
    "data": [],
    "layout": {
        "title": "",
        "xaxis": { "title": "" },
        "yaxis": { "title": "" }
    }
}


After adding hints, Record.print_to_inspect() is expected to give the below output.
{
    "comments": "Use Record.set_comments() to populate this field...",
    "datatype": "Use Record.set_datatype() to populate this field...",
    "data": [],
    "layout": {
        "title": "Use Record.set_graph_title() to populate this field...",
        "xaxis": { "title": "Use Record.set_x_axis_label() to populate this field..." },
        "yaxis": { "title": "Use Record.set_y_axis_label() to populate this field..." }
    }
}

Step 3: Populating Fields

Fill in the required fields for the graph.

Record.set_comments("Tree Growth Data collected from the US National Arboretum")
Record.set_datatype("Tree_Growth_Curve")
Record.set_x_axis_label_including_units(x_label_including_units)
Record.set_y_axis_label_including_units(y_label_including_units)
Record.add_data_series(series_name="pear tree growth", x_values=time_in_years, y_values=tree_heights, plot_type="scatter_spline")
Record.set_graph_title("Pear Tree Growth Versus Time")
Record.update_plot_types()

Expected Output:

Warning: Your x axis label did not pass expected validation checks...
["The units of years appear to be plural. Units should be entered as singular, such as 'year' rather than 'years'.",
"Now removing the 's' to change the units into singular 'year'..."]

Step 4: Exporting to File

Save the JSON record for use in JSONGrapher.

Record.remove_hints()
filename_to_export_to = "./ExampleFromTutorial.json"
Record.export_to_json_file(filename_to_export_to)
print("JSONGrapher Record exported to " + filename_to_export_to)
#Let's print the final record:
Record.print_to_inspect()

Expected Output:

JSONGrapher Record exported to, ./ExampleFromTutorial.json
{
    "comments": "Tree Growth Data collected from the US National Arboretum",
    "datatype": "Tree_Growth_Curve",
    "data": [
        {
            "name": "pear tree growth",
            "x": [0, 1, 2, 3, 4],
            "y": [0, 0.42, 0.86, 1.19, 1.45],
            "type": "scatter",
            "line": { "shape": "spline" }
        }
    ],
    "layout": {
        "title": "Pear Tree Growth Versus Time",
        "xaxis": { "title": "Time (year)" },
        "yaxis": { "title": "Height (m)" }
    }
}

Plot and export the graph using Matplotlib.
